# Array Reverse 

Code Challenge #01 : Pair programmed with Ben Hill 

# Challenge

Take an array and reverse it.
- In Place

# Algorithm

- Iterate the array, noting the start and end
- Swap them
- Move each of them in 1 place
  - Continue to swap
- When at the middle, stop
